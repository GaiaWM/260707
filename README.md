# 260707 — The Ghost in the Shell

A presentation on the idea behind **GaiaWM** and its embodied agents:
an LLM **ghost** in an eight-organ MCP **shell** — energy, memory, skills,
abilities, inventory, comms, world, calibration — grounded in a geospatial
world model and measured against a world that pushes back.

*Cognition is not the model in isolation; it is the interaction of an
inference engine with a structured, embodied substrate.*

**View the deck:** <https://gaiawm.github.io/260707/>

Navigation: ← → · space · click the screen edges. Slide 10 carries live
calibration data from the Toril simulation (overconfidence grows with plan
depth: +0.04 → +0.18 → +0.32 across plan orders 1–3, n = 600).
